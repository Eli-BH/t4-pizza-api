## Description

This project is a NestJS-based API for managing pizza orders. It uses NestJS and TypeORM with a PostgreSQL database 

## Project setup

```bash
$ npm install
```

## Run Docker container
```bash
$ docker-compose up -d
```

## Compile, Seed Database, and run the project 

```bash

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## If database is not found
Stop your local postgres service and run docker container again 
```bash
$ docker-compose down -v
$ docker-compose up -d
```

## Run tests

```bash
# e2e tests
$ npm run test:e2e
```
